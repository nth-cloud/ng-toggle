Manual installation

If you prefer not to use schematics or want to add ng-toggle to an older project, you'll need to do the following:

  • Add bootstrap and @nth-cloud/ng-toggle dependencies from npm
  • Add bootstrap CSS/SCSS to your project (no javascript is required)
  • Import NgToggleModule, or directly import the ng-toggle components and directives that you use, which are standalone.

Please, do not open issues for the general support questions as we want to keep GitHub issues for bug reports and feature requests. You've got much better chances of getting your question answered on StackOverflow where the community at large are looking at questions tagged with ng-toggle.

StackOverflow is a much better place to ask questions since:

  • there are hundreds of people willing to help on StackOverflow,
  • questions and answers stay available for public viewing so your question / answer might help someone else, and
  • the SO voting system assures that the best answers are prominently visible.
